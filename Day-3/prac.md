Create a Node.js program using only Sync methods of the File System (fs) module.

Requirements
Create a folder named StudentRecords.
Inside the StudentRecords folder, create the following files:
Rahul.txt
Priya.txt
Aman.txt
Write the following data into each file.
Rahul.txt
Name: Rahul
Course: Node.js
City: Indore
Priya.txt
Name: Priya
Course: React.js
City: Bhopal
Aman.txt
Name: Aman
Course: Express.js
City: Jaipur
Read all the files one by one and display their content on the console.
Create a new file named allStudents.txt inside the same folder.
Write the following student names into allStudents.txt.
Rahul
Priya
Aman
Read and display the content of allStudents.txt.
Rename allStudents.txt to studentsList.txt.
Check whether studentsList.txt exists or not.
Finally, display all files present inside the StudentRecords folder.
